# VulnerableCode Insights Dashboard

## Overview
This project extends VulnerableCode by adding an insights module that provides analytical views of vulnerability data.

## Features
- Vulnerabilities per year API
- Severity distribution API
- Dashboard visualization using Chart.js

## Motivation
VulnerableCode currently lacks visualization tools for understanding vulnerability trends. This project aims to fill that gap.
